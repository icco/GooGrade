LemmingScheduler Daily Builds

This process is entirely automated with no human interaction. Hopefully, removing humans from this process will create a more reliable daily build system.

Builds are done at 04:00 Pacific. If no new build appears that means no code has been committed since the last automatic build. Clicking the revision number shows a change log containing svn commit messages since the last daily build.

WARNING: All of these builds are considered unstable until an official release is made.
